# WasteCare PWA Icons

This directory contains the PWA icons for WasteCare application.

## Icon Specifications:
- **Primary Color**: #16a34a (Green)
- **Background**: White (#ffffff)
- **Theme**: Waste management/environmental

## Shortcut Icons:
- shortcut-report.png (96x96) - For reporting waste
- shortcut-map.png (96x96) - For viewing map

## Design Guidelines:
1. Use the green theme color (#16a34a)
2. Include waste/recycling symbols
3. Ensure good contrast and readability
4. Test on various backgrounds
5. Make icons work as maskable icons

- PWA Asset Generator
